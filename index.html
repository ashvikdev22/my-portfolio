Out-of-School Product Development — 2025

Mandai Rainforest Resort
Nature Walk App

Co-developed a responsive virtual nature walk application tailored entirely for Mandai Rainforest Resort. Because this was an entirely new, unmapped architectural landscape, our team handled the project end-to-end: conducting independent on-site tracking to collect precise GPS coordinates, prototyping custom mapping pipelines, and implementing open-source map APIs to construct a seamless, interactive user journey.

The final system bridges the gap between field exploration and modern software interface architecture, receiving an official Certificate of Appreciation from the Banyan Tree General Manager.

SST Capstone Project × DSO National Laboratories — 2025/2026

Autonomous Sound-Countermeasure
& Patrol Robot

Currently engineering an advanced architectural hardware-software prototype for an autonomous patrolling robotic system. Developed alongside security specialists at DSO National Laboratories, this robot is designed to operate seamlessly within contested environments to counter specialized radio and jamming signals generated by opposition forces.

The logic loop integrates real-world navigation algorithms, automated situational analysis routines, and adaptive transmission matrices, enabling the unit to localize jamming sources and dynamically deploy signal countermeasures to ensure secure tactical network channels.
